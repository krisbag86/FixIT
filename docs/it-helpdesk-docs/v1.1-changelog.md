# FixIT v1.1 - changelog roboczy

Ten dokument zbiera zmiany przygotowane na branchu `v-1.1`.
9 czerwca 2026 branch zostal zmergowany do `main` i wdrozony na Railway.

## Zmiany

### 2026-06-05 - Ikona w pasku przeglądarki

- Dodano `app/icon.svg` jako faviconę aplikacji FixIT.
- Podpięto ikonę w metadata aplikacji przez `app/layout.tsx`.
- Zweryfikowano zmianę komendami `npm run typecheck` i `npm run build`.

### 2026-06-09 - Rejestracja i onboarding użytkowników

- Dodano publiczną rejestrację `/register` tylko dla adresów w domenie `bagietka.pl`.
- Samodzielna rejestracja tworzy konto z rolą `REPORTER` i od razu loguje użytkownika.
- Rozszerzono `/admin/users` o tworzenie nowych kont, jednorazowe linki aktywacyjne i opcjonalną wysyłkę e-mailem.
- Konta tworzone przez admina wymuszają zmianę hasła przy pierwszym logowaniu.

### 2026-06-09 - Railway

- Zmergowano branch `v-1.1` do `main`.
- Wypchnięto `main` do GitHub i potwierdzono start auto-deployu na Railway.
